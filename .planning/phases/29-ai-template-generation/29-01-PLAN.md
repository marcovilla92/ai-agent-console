Add Pydantic models:

```python
class GenerateTemplateRequest(BaseModel):
    description: str  # Natural language project description (max 2000 chars)
    stack: str | None = None  # Optional stack hint

class GenerateTemplateResponse(BaseModel):
    id: str
    name: str
    description: str
    files: dict[str, str]
    validation_errors: list[str]
```

Add validation helper `_validate_generated_files(files: dict[str, str]) -> list[str]`:
- Create temp directory with `tempfile.mkdtemp(prefix="tmpl-validate-")`
- For each file in `files`: check path safety (reject paths starting with `/` or containing `..`), write to temp dir
- Check `.claude/agents/` files for PROTECTED_AGENTS name collisions (stem, lowercased, spaces to hyphens)
- Run `discover_project_agents(tmp_dir)` wrapped in try/except to catch parse errors
- Run `discover_project_commands(tmp_dir)` wrapped in try/except to catch parse errors
- Always `shutil.rmtree(tmp_dir, ignore_errors=True)` in finally block
- Return list of error strings (empty = valid)

Add endpoint `@template_router.post("/generate", response_model=GenerateTemplateResponse)`:
- Check `_gen_lock.locked()` first -- if True, raise HTTPException 429 with detail "Template generation already in progress" and header `Retry-After: 30`
- Use `async with _gen_lock:` for the generation block
- Cap `req.description` to 2000 chars
- Build prompt: `f"Generate a project template for: {req.description}"` + optional `f"\nPreferred stack: {req.stack}"` if stack provided
- Load system prompt text from `_GEN_SYSTEM_PROMPT_PATH` using `Path.read_text()`
- Call `await call_orchestrator_claude(prompt=prompt, schema=TEMPLATE_GEN_SCHEMA, system_prompt=system_prompt_text)`
- Parse response using the orchestrator pattern: `json.loads(raw)` -> `.get("structured_output") or .get("result")` -> if string, `json.loads(data)`
- If parsing fails, raise HTTPException 502 with detail "AI generation produced invalid response"
- Run `_validate_generated_files(data.get("files", {}))`
- Return GenerateTemplateResponse with all fields

Add `asyncio.wait_for(..., timeout=120)` around the `call_orchestrator_claude` call. Catch `asyncio.TimeoutError` and raise HTTPException 504 with detail "Template generation timed out".

Use logging (`log = logging.getLogger(__name__)` -- add if not already present) to log generation requests and results.
