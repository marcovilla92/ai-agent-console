1. **Add response model to `src/server/routers/tasks.py`:**

   Add an `AgentOutputResponse` Pydantic model:
   ```python
   class AgentOutputResponse(BaseModel):
       id: Optional[int] = None
       agent_type: str
       raw_output: str
       created_at: datetime

   class AgentOutputListResponse(BaseModel):
       outputs: list[AgentOutputResponse]
       count: int
   ```

2. **Add the endpoint to `src/server/routers/tasks.py`:**

   Import `get_pool` from `src.server.dependencies` and `AgentOutputRepository` from `src.db.pg_repository`.

   Add a new route AFTER the existing `get_task` route:
   ```python
   @task_router.get("/{task_id}/outputs", response_model=AgentOutputListResponse)
   async def get_task_outputs(
       task_id: int,
       pool: asyncpg.Pool = Depends(get_pool),
   ):
       """Get agent output history for a task."""
       repo = AgentOutputRepository(pool)
       outputs = await repo.get_by_session(task_id)
       return AgentOutputListResponse(
           outputs=[
               AgentOutputResponse(
                   id=o.id,
                   agent_type=o.agent_type,
                   raw_output=o.raw_output,
                   created_at=o.created_at,
               )
               for o in outputs
           ],
           count=len(outputs),
       )
   ```

   Import `asyncpg` at the top of the file. Note: the endpoint returns an empty list if no outputs exist (not 404) -- this is correct behavior since a task may have no agent outputs yet.
