Key dialect differences from aiosqlite (do NOT use SQLite syntax):
- Placeholders: `$1, $2, $3` not `?`
- Auto-increment: `SERIAL PRIMARY KEY` not `INTEGER PRIMARY KEY AUTOINCREMENT`
- Timestamps: `TIMESTAMPTZ` not `TEXT`
- Floats: `DOUBLE PRECISION` not `REAL`
- Return IDs: `RETURNING id` with `fetchval()` not `cursor.lastrowid`
