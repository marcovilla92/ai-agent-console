    **Step 3 — Create the template router** at `src/server/routers/templates.py`:

    Module-level constants:
    ```python
    TEMPLATES_ROOT = Path(__file__).resolve().parent.parent.parent.parent / "templates"
    REGISTRY_PATH = TEMPLATES_ROOT / "registry.yaml"
    EXCLUDE_DIRS = {".git", "__pycache__", "node_modules", ".mypy_cache"}
    ```

    Pydantic models:
    - `TemplateFile(BaseModel)`: path (str), type (str — "jinja2" or "static"), size (int)
    - `TemplateSummary(BaseModel)`: id (str), name (str), description (str), builtin (bool)
    - `TemplateListResponse(BaseModel)`: templates (list[TemplateSummary])
    - `TemplateDetail(BaseModel)`: id (str), name (str), description (str), builtin (bool), files (list[TemplateFile])

    Helper functions:
    - `load_registry() -> dict`: `yaml.safe_load(REGISTRY_PATH.read_text())` or `{"templates": []}` if empty
    - `save_registry(data: dict) -> None`: `yaml.safe_dump` to REGISTRY_PATH
    - `get_file_manifest(template_id: str) -> list[TemplateFile]`: Walk `TEMPLATES_ROOT / template_id` with `rglob("*")`, filter files only, exclude EXCLUDE_DIRS, return path (relative), type (.j2 = "jinja2", else "static"), size

    Endpoints:
    - `GET /templates` — load registry, return TemplateListResponse
    - `GET /templates/{template_id}` — find entry in registry (404 if missing), build file manifest, return TemplateDetail
