    **Step 3 — Add `safe_write_template_file` helper:**
    ```python
    def safe_write_template_file(template_dir: Path, rel_path: str, content: str) -> None:
        target = (template_dir / rel_path).resolve()
        if not target.is_relative_to(template_dir.resolve()):
            raise ValueError(f"Path traversal attempt: {rel_path!r}")
        target.parent.mkdir(parents=True, exist_ok=True)
        target.write_text(content, encoding="utf-8")
    ```

    **Step 4 — Add `_get_entry_or_404` helper** that finds a registry entry by id and raises 404 if missing:
    ```python
    def _get_entry_or_404(data: dict, template_id: str) -> dict:
        entry = next((t for t in data["templates"] if t["id"] == template_id), None)
        if entry is None:
            raise HTTPException(status_code=404, detail="Template not found")
        return entry
    ```

    **Step 5 — Implement POST /templates:**
    1. Load registry, check id does not already exist (409 Conflict if duplicate)
    2. Create `TEMPLATES_ROOT / request.id` directory
    3. For each file in `request.files`, call `safe_write_template_file` (catch ValueError -> 400)
    4. Append entry to registry with `builtin: false`
    5. Save registry
    6. On any error after dir creation, clean up with `shutil.rmtree`
    7. Return 201 with TemplateCreateResponse

    6. Save registry
    7. Return 200 with TemplateCreateResponse (reuse model: id, name, description, builtin, file_count)
