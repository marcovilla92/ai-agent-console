**Step 3: Color palette and design tokens**

Use these Tailwind classes consistently throughout:

- **Background:** `bg-surface-50` (page), `bg-white` (cards/surfaces)
- **Text:** `text-surface-900` (headings), `text-surface-700` (body), `text-surface-500` (muted/secondary)
- **Primary actions:** `bg-primary-600 hover:bg-primary-700 text-white` (buttons)
- **Secondary actions:** `bg-white border border-surface-300 text-surface-700 hover:bg-surface-50`
- **Danger actions:** `bg-red-600 hover:bg-red-700 text-white`
- **Status colors:** running=`text-sky-600 bg-sky-50`, completed=`text-emerald-600 bg-emerald-50`, failed=`text-red-600 bg-red-50`, queued=`text-surface-500 bg-surface-100`, awaiting_approval=`text-amber-600 bg-amber-50`, cancelled=`text-surface-400 bg-surface-100`
- **Typography:** Use default Tailwind sans. Headings: `text-2xl font-bold text-surface-900`. Subheadings: `text-sm text-surface-500`.
- **Spacing:** `space-y-6` between major sections, `p-6` card padding, `gap-4` for grids
