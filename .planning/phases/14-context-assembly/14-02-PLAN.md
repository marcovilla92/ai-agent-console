Create src/server/routers/projects.py following the exact pattern from templates.py router:

1. Pydantic response models:
   - RecentTaskResponse(BaseModel): id (int), prompt (str), status (str), created_at (str)
   - ContextResponse(BaseModel): workspace (str), claude_md (str), planning_docs (dict[str, str]), git_log (str), recent_tasks (list[RecentTaskResponse])
   - PhaseSuggestion(BaseModel): phase_id (str), phase_name (str), status (str), reason (str)
   - PhaseSuggestionResponse(BaseModel): suggestion (PhaseSuggestion | None), all_phases (list[dict])

2. Router setup:
   ```python
   project_router = APIRouter(
       prefix="/projects",
       tags=["projects"],
       dependencies=[Depends(verify_credentials)],
   )
   ```

3. GET /{project_id}/context endpoint:
   - Depends(get_pool) for pool
   - ProjectRepository(pool).get(project_id) -- raise 404 if None
   - await assemble_full_context(project.path, pool)
   - Return ContextResponse

4. GET /{project_id}/suggested-phase endpoint:
   - Depends(get_pool) for pool
   - ProjectRepository(pool).get(project_id) -- raise 404 if None
   - await suggest_next_phase(project.path)
   - Return PhaseSuggestionResponse (suggestion=None if no .planning/)

Then update src/server/app.py:
- Import project_router from src.server.routers.projects
- Add app.include_router(project_router) after template_router
