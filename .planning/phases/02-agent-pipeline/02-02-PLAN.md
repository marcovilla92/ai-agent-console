# Plan 02-02: PLAN/EXECUTE/REVIEW Agent Implementations

## Goal
Implement the three concrete agents with their system prompts and output contracts (AGNT-01, AGNT-02, AGNT-03).

## Tasks

1. Create `src/agents/prompts/` directory with system prompt templates:
   - `plan_system.txt`: Instructs PLAN agent to output GOAL, ASSUMPTIONS, CONSTRAINTS, TASKS, ARCHITECTURE, FILES TO CREATE, HANDOFF sections
   - `execute_system.txt`: Instructs EXECUTE agent to output TARGET, PROJECT STRUCTURE, FILES, CODE, COMMANDS, SETUP NOTES, HANDOFF sections
   - `review_system.txt`: Instructs REVIEW agent to output SUMMARY, ISSUES, RISKS, IMPROVEMENTS, DECISION sections

2. Update agent configs in `src/agents/config.py`:
   - Each config references its prompt template path
   - Each config defines expected output_sections matching the contract

3. Create `src/agents/plan.py`, `src/agents/execute.py`, `src/agents/review.py`:
   - Thin subclasses of BaseAgent (only if agent-specific logic needed)
   - Factory function: `create_agent(name: str, db, project_path) -> BaseAgent`

4. Tests:
   - Test each agent's config has correct output_sections
   - Test system prompt templates exist and contain key instructions
   - Test factory creates correct agent types
