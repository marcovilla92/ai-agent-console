2. Create `src/engine/context.py` with `WebTaskContext`:
   - Constructor: `__init__(self, task_id: int, pool: asyncpg.Pool, mode: str, project_path: str = ".")`
   - `project_path` property returning `self._project_path`
   - `update_status()`: no-op in Phase 7 (just log). WebSocket streaming is Phase 8.

3. Create `src/engine/manager.py` with `TaskManager`:
   - `__init__(self, pool: asyncpg.Pool, max_concurrent: int = 2)`: Store pool, create `asyncio.Semaphore(max_concurrent)`, init `self._running: dict[int, RunningTask]` dict.
   - `RunningTask` dataclass: `handle: asyncio.Task`, `task_id: int`, `ctx: WebTaskContext`.
   - `async def submit(self, prompt: str, mode: str = "autonomous", project_path: str = ".") -> int`: Insert task row via TaskRepository (name = first 50 chars of prompt), create asyncio.Task wrapping `_execute`, store in `_running`.
   - `async def _execute(self, task_id: int, prompt: str, mode: str, project_path: str) -> None`: Set status "queued", acquire semaphore (`async with`), set status "running", create WebTaskContext, call `orchestrate_pipeline(ctx, prompt, pool, task_id)`, set status "completed" or "failed". On CancelledError set "cancelled". Always update status in finally block and remove from `_running`.
   - `async def cancel(self, task_id: int) -> bool`: Look up in `_running`, call `handle.cancel()`, terminate subprocess via `ctx.proc` if set (SIGTERM then SIGKILL after 5s timeout). Return True if found.
   - `async def get(self, task_id: int) -> Optional[Task]`: Delegate to TaskRepository.get.
   - `async def list_all(self) -> list[Task]`: Delegate to TaskRepository.list_all.
   - `async def shutdown(self) -> None`: Cancel all running tasks (for clean server shutdown).
