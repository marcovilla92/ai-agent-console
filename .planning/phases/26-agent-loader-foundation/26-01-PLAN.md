  <action>
Create `src/agents/loader.py` with:

1. `CLAUDE_AGENTS_DIR = ".claude/agents"` constant.

2. `discover_project_agents(project_path: str) -> dict[str, AgentConfig]`:
   - Build path: `Path(project_path) / CLAUDE_AGENTS_DIR`
   - If directory doesn't exist, return `{}`
   - Iterate `sorted(agents_dir.glob("*.md"))` for deterministic order
   - For each file, call `_parse_agent_md(md_file)` in try/except, log warning on failure, skip
   - Return dict keyed by agent name

3. `_parse_agent_md(md_path: Path) -> AgentConfig | None`:
   - Read file with `encoding="utf-8", errors="replace"`
   - Skip empty files (return None with warning)
   - Parse with `frontmatter.loads(content)` -- `post.metadata` is dict (empty if no frontmatter), `post.content` is body
   - Build AgentConfig with:
     - `name`: from metadata `name` key, or `_sanitize_name(md_path.stem)`
     - `system_prompt_file`: `""` (project agents use inline)
     - `system_prompt_inline`: `post.content` (the markdown body)
     - `description`: from metadata, or `f"Project agent: {name}"`
     - `output_sections`: from metadata, or `[]`
     - `next_agent`: from metadata, or `None`
     - `allowed_transitions`: `tuple(meta.get("allowed_transitions", ("plan", "execute", "test", "review", "approved")))` -- broad default
     - `source`: `"project"`
     - `file_path`: `str(md_path.resolve())`

4. `_sanitize_name(stem: str) -> str`:
   - `stem.lower().replace(" ", "-")` then `re.sub(r"[^a-z0-9-]", "", result)` then strip leading/trailing hyphens

Use `import logging; log = logging.getLogger(__name__)` at module level.

Create `tests/test_agent_loader.py` with pytest tests using `tmp_path` fixture to create temporary `.claude/agents/` directories with test .md files. Write tests FIRST (red), then implement (green).
